Android Workshop

This workshop is a two day Android Mobile Application Development program. This workshop will introduce the students to Android (www.android.com), which is an open-source operating system. The demand for Android app development has grown through its robust offerings with a plethora of new android devices. This provides a great deal of opportunities for students to explore, understand and further their knowledge base by leaps and bounds. The primary focus of this workshop is to enable the students to develop Android applications on their own. Android is presently the world's best selling Smartphone platform.

Android Workshop is a program designed to demystify mobile application development. The two day program is streamlined in such a way that beginners with no prior programming experience will be equipped to build their own apps by the end of the workshop.


Workshop Features:

  • Hands-on training on developing mobile applications using JAVA

  • Hands-on exposure to using drag & drop method to build exciting apps

  • Build your own mobile apps in the workshop itself!

  • Participation certificate issued

  • Software CD with resources and Apps


Concepts covered:

  • Introduction and basics of Android.

  • Mobile OS Family

  • Selection Criteria for Android

  • Architecture of Android

  • Android UI

  • Dalvik virtual machine optimized for mobile

  • Core Java introduction and hands-on

  • XML basics

  • Android APIs

  • Android application building using Android Studio

  • Android application building using drag & drop method

  • Sensor based apps

  • Android in marketplace


Workshop fee:

  • Rs. 1800/- for a team of two.
